# Smart Mirror IoT

Proiect IoT realizat cu ESP32 pentru controlul inteligent al iluminării unei oglinzi.

## Funcționalități
- Detectarea prezenței utilizatorului cu senzor PIR
- Reglarea automată a luminozității în funcție de lumina ambientală (LDR)
- Control manual prin buton capacitiv
- Control de la distanță prin Bluetooth
- Iluminare cu bandă LED WS2812B

## Hardware utilizat
- ESP32
- PIR Sensor HC-SR501
- LDR
- Capacitive touch sensor TTP223
- LED strip WS2812B

## Software
- Arduino IDE
- FastLED library
- BluetoothSerial
